## Cara Menambahkan Gambar

### 1. Icon Surat (Amplop)
**Lokasi**: `public/assets/icons/surat.png`

**Cara:**
1. Siapkan gambar icon surat/amplop (PNG dengan background transparan)
2. Copy file ke: `public/assets/icons/surat.png`
3. Refresh halaman

### 2. Background Popup
**Lokasi**: `public/assets/images/bg/background-popup.png`

**Cara:**

## ðŸ’¡ Tips Gambar

### Icon Surat
- **Ukuran**: 200x200px atau lebih
- **Format**: PNG (dengan transparent background)
- **Style**: Cute, simple, romantic
- **Contoh**: Amplop pink, surat dengan hati, envelope dengan pita

### Background Popup
- **Ukuran**: 800x600px atau lebih besar
- **Format**: PNG atau JPG
- **Style**: Soft, tidak terlalu ramai
- **Contoh**: 
  - Paper texture dengan watermark hearts
  - Subtle floral pattern
  - Soft gradient dengan ornamen
  - Vintage letter paper

## ðŸš¨ Troubleshooting

**Icon tidak muncul?**
- Cek path: `public/assets/icons/surat.png`
- Cek permission folder
- Refresh browser (Ctrl + F5)
- Lihat console browser untuk error

**Background tidak muncul?**

- Sistem akan tetap bagus dengan fallback gradient
